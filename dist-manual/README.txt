# BlueChip AI Browser - Manual Distribution

## Quick Start

1. **Double-click `start.bat`** to launch the application
2. **Or run manually**: `npx electron electron/main-browser.js`

## Requirements

- **Node.js** (version 18 or higher)
- **npm** (comes with Node.js)

## Installation

1. Install Node.js from https://nodejs.org/
2. Open Command Prompt/PowerShell in this folder
3. Run: `npm install` (if needed)
4. Double-click `start.bat` to start the browser

## Features

- ✅ AI-powered browsing with Google Gemini integration
- ✅ Modern tab management
- ✅ Bookmark and history management
- ✅ Secure browsing with input sanitization
- ✅ SQLite database for data persistence

## Troubleshooting

- **App won't start**: Make sure Node.js is installed
- **AI features not working**: Check your Gemini API key in settings
- **Database errors**: Ensure the app has write permissions
